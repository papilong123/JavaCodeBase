### 3. 要求  
   Spring Data Redis 2。x二进制文件需要JDK 8.0及以上，Spring Framework 5.3.9及以上
   在redis版本方面，Redis 2.6.X或更高。Spring Data Redis目前针对最新的4.0版本进行测试
